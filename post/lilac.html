lilac- a static blog generator that sucks less

May 25, 13

lilac

It is - a static blog generator

Demo

demo site is on branch gh-pages

  • http://hit9.org/lilac

Featrues

  • Toml & Markdown(GFM) based

  • 100% in Python

  • tags & feed & templates & Code highlighting support

  • jinja2 templating language

  • minimal blog

Quick Start

  1. install lilac:
mkdir MyBlog && cd MyBlog
virtualenv venv
. venv/bin/activate
pip install git+git://github.com/hit9/lilac.git
  1. deploy your blog:
lilac deploy

lilac generate you a Makefile to manage blog.

  1. edit config.toml to meet your needs.

  2. build blog:

make build
  1. see your blog in action:
make server

preview htmls at http://0.0.0.0:8000

Write a Post

To write a post, just follow the 3 simple steps:

  1. new a file in directory src/post , for example: src/post/hello-world.md

  2. edit that file, write header in Toml and write body in markdown, header and body are separated with a line ----

  3. run make build , and then preview under the ''cute' server in your browser

OS Plat

Only works on *nix.

windows doesn't count.

FAQ?

  • Don't forget to activate virtualenv environment - . venv/bin/activate before building
